# Workers benchmark

Deploy minimum worker code for JavaScript and Rust(wasm) and compare them.

Original code are generated by

- `wrangler generate`
- `wrangler generate type=rust`

and align implementation details.

## Code
### javascript
```javascript
addEventListener("fetch", (ev) => {
  ev.respondWith(handleEvent(ev))
})

async function handleEvent() {
  return new Response("Hello worker!");
}
```

### rust
```rust
use worker::*;

#[event(fetch)]
pub async fn main(_req: Request, _env: Env, _ctx: worker::Context) -> Result<Response> {
    Response::ok("Hello worker!")
}
```
